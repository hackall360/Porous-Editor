POROUS EDITOR

Version 1.0.0

A fully-featured, 100% client-side save file editor. Porous Editor supports multiple game save formats and provides an intuitive interface for editing game saves directly in your browser.

FEATURES

  • 100% Client-Side - All processing happens in your browser. No data is uploaded.
  • Multi-Format Support - JSON/Unity, RPG Maker series, and raw files
  • Real-Time Editing - Instant preview with live updates
  • Type-Safe - Built with TypeScript for reliability
  • Responsive Design - Works on desktop and mobile
