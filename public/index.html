SUPPORTED FORMATS

Porous Editor can load and edit these save file formats

JSON / Unity

.json, .save

Structured JSON files from Unity games, Ren'Py visual novels, and other engines that save as JSON.

Editing Mode: Structured Editor
Auto-detects money/gold fields
Parses inventory items
Edits numeric/string variables

RPG Maker MV/MZ

.rmmzsave, .rpgsave

Save files from RPG Maker MV (2015) and RPG Maker MZ (2020). These are binary files that store game data.

Editing Mode: Raw Text Editor
Opens as raw text/binary
Use "Format JSON" if it contains JSON
Be careful with binary data

RPG Maker VX Ace/XP

.rvdata2, .rxdata

Save files from RPG Maker VX Ace (2012) and RPG Maker XP (2004). Uses Ruby Marshal format.

Editing Mode: Raw Text Editor
Binary Ruby Marshal format
May contain readable strings
Direct editing can corrupt saves

RPG Maker 2000/2003

.lsd

Legacy save files from RPG Maker 2000 and 2003. Uses a custom binary format.

Editing Mode: Raw Text Editor
Very old binary format
Limited direct editing possible
Use specialized tools for best results

Generic Saves

.sav (various)

Generic save files from many games including Wolf RPG, Unreal Engine, and many others.

Editing Mode: Raw Text Editor
Format varies by game
May be binary or text
Try "Format JSON" if unsure

Other Formats

Any extension

Any file type not explicitly listed will open in raw mode for direct text/binary editing.

Editing Mode: Raw Text Editor
Full text editor with syntax
"Format JSON" button for JSON data
Clear and restart if needed

ABOUT EDITING MODES

Structured Editor

For JSON-based saves. Provides dedicated inputs for money, inventory items, and variables. Changes are applied intelligently with type preservation. Safer and more user-friendly.

Raw Text Editor

For binary or unknown formats. Full textarea with raw content editing. Includes "Format JSON" button to pretty-print JSON data. Use with caution - can corrupt binary saves.

Tip: If your save format isn't listed, try opening it anyway - Porous Editor will use raw mode and you can still edit text-based content.

Formats are automatically detected by file extension. If a format isn't working, make sure your file has the correct extension.
