POROUS EDITOR

Version 1.0.0

A fully-featured, 100% client-side save file editor. Porous Editor supports multiple game save formats and provides an intuitive interface for editing game saves directly in your browser.

100% Client-Side

All processing happens in your browser. No save files are uploaded to any server. Your data never leaves your device - complete privacy and security.

Multi-Format Support

Supports JSON/Unity saves, RPG Maker series (MV/MZ/VX/2000/2003), and generic save files. Automatically detects format by extension.

Real-Time Editing

See changes instantly as you edit. No need to save and reload. The interface updates immediately with live preview of your edits.

Type-Safe

Built with TypeScript for maximum reliability. Comprehensive type checking ensures fewer bugs and better code quality. Strict mode enabled throughout.

Responsive Design

Works seamlessly on desktop and mobile devices. The interface adapts to your screen size with a modern, intuitive design.

Easy Export

Download your edited saves with original filenames preserved. One-click download with proper file extensions and MIME types.
