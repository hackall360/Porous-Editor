SUPPORTED FORMATS

Porous Editor can load and edit these save file formats. Each format uses either the Structured Editor (for JSON) or Raw Text Editor (for binary/unknown formats).

JSON / Unity

.json, .save

Structured JSON files from Unity games, Ren'Py visual novels, and other engines that save as human-readable JSON.

Editing Mode: Structured Editor

Auto-detects money/gold fields

Parses inventory items

Edits numeric/string variables

Type-safe with validation

RPG Maker MV/MZ

.rmmzsave, .rpgsave

Save files from RPG Maker MV (2015) and RPG Maker MZ (2020). These are binary files that store game data in a proprietary format.

Editing Mode: Raw Text Editor

Opens as raw text/binary

Use "Format JSON" if it contains JSON

Be careful with binary data

May need specialized tools for full editing

RPG Maker VX Ace/XP

.rvdata2, .rxdata

Save files from RPG Maker VX Ace (2012) and RPG Maker XP (2004). Uses Ruby Marshal format for serialization.

Editing Mode: Raw Text Editor

Binary Ruby Marshal format

May contain readable strings

Direct editing can corrupt saves

Use with extreme caution

RPG Maker 2000/2003

.lsd

Legacy save files from RPG Maker 2000 and 2003. Uses a custom binary format that is quite old and not well documented.

Editing Mode: Raw Text Editor

Very old binary format

Limited direct editing possible

Use specialized tools for best results

Consider using dedicated RPG Maker editors

Generic Saves

.sav (various)

Generic save files from many games including Wolf RPG, Unreal Engine, and many others. Format varies significantly by game.

Editing Mode: Raw Text Editor

Format varies by game

May be binary or text

Try "Format JSON" if unsure

Some may work better with other tools

Other Formats

Any extension

Any file type not explicitly listed will open in raw mode for direct text/binary editing. Porous Editor will attempt to detect if it's JSON.

Editing Mode: Raw Text Editor

Full text editor with syntax

"Format JSON" button for JSON data

Clear and restart if needed

Return to upload for different file

ABOUT EDITING MODES

Structured Editor

For JSON-based saves. Provides dedicated inputs for money, inventory items, and variables. Changes are applied intelligently with type preservation. Safer and more user-friendly - you don't need to understand JSON structure to edit values.

Raw Text Editor

For binary or unknown formats. Full textarea with raw content editing. Includes "Format JSON" button to pretty-print JSON data if the file contains JSON. Use with caution - editing binary data incorrectly can corrupt saves.

Tip: If your save format isn't listed, try opening it anyway - Porous Editor will use raw mode and you can still edit any text-based content. Some formats may work better than others.

Format Detection: Formats are automatically detected by file extension. If a format isn't working, make sure your file has the correct extension for that game/engine.

Safety: Always back up your original save files before editing. Porous Editor works 100% client-side - your files never leave your browser.
