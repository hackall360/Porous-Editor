About Editing Modes

Structured Editor

For JSON-based and parser-supported formats. Provides dedicated inputs for common fields like money, inventory, and variables. Changes are applied intelligently with type preservation. Safer and more user-friendly than raw editing.

Raw Text Editor

For binary or unknown formats. Full textarea with raw content editing. Includes "Format JSON" button to pretty-print JSON data. Use with caution - direct editing of binary formats can corrupt saves. Best for text-based formats or advanced users.

Tip: If your save format isn't explicitly listed, try opening it anyway - Porous Editor will use raw mode and you can still edit any text-based content. Many formats are detected automatically by file header (magic bytes), not just extension.

Format Detection: Formats are detected by file extension first, then by header magic bytes if available. If a format isn't working, ensure your file has the correct extension or try renaming it to match a supported format.

POROUS EDITOR

Version 1.0.0

A fully-featured, 100% client-side save file editor. Porous Editor supports multiple game save formats and provides an intuitive interface for editing game saves directly in your browser.

FEATURES

  • 100% Client-Side - All processing happens in your browser. No data is uploaded.
  • Multi-Format Support - JSON/Unity, RPG Maker series, and raw files
  • Real-Time Editing - Instant preview with live updates
  • Type-Safe - Built with TypeScript for reliability
  • Responsive Design - Works on desktop and mobile
